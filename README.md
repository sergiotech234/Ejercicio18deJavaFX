# Formulario con Múltiples Botones en Java

## Descripción

Este proyecto consiste en desarrollar una aplicación en **Java** con una interfaz gráfica que contiene **varios botones**. Cada botón ejecuta una acción distinta y muestra un mensaje diferente en la consola al ser pulsado.

El objetivo del ejercicio es practicar el manejo de eventos y la interacción entre componentes gráficos utilizando **Swing**.

## Objetivos

- Crear una interfaz gráfica con varios botones.
- Implementar eventos independientes para cada botón.
- Mostrar mensajes distintos en la consola según el botón pulsado.
- Familiarizarse con el manejo de eventos en Java.

## Funcionamiento

1. Se inicia la aplicación.
2. Aparece una ventana con varios botones.
3. El usuario pulsa uno de los botones disponibles.
4. Cada botón ejecuta una acción distinta.
5. Se muestra un mensaje específico en la consola.


## Ejemplo de uso

Ventana principal:

```text
+--------------------------------+
|                                |
|   [ Botón 1 ] [ Botón 2 ]      |
|                                |
|   [ Botón 3 ]                  |
|                                |
+--------------------------------+
```

Salida en consola:

Si se pulsa **Botón 1**

```text
Has pulsado el botón 1
```

Si se pulsa **Botón 2**

```text
Has pulsado el botón 2
```

Si se pulsa **Botón 3**

```text
Has pulsado el botón 3
```
